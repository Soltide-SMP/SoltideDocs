# How to Join



{% hint style="info" %}
Bedrock support is currently in beta.\
Soltide only supports Bedrock clients on Windows, iOS, and Android.
{% endhint %}

{% tabs %}
{% tab title="Java" %}
{% stepper %}
{% step %}
### Enter the Multiplayer screen

Select Multiplayer in your start menu
{% endstep %}

{% step %}
### Select <kbd>Add Server</kbd>

Fill Soltide in as the name and <kbd>Play.Soltide.net</kbd> for the IP section
{% endstep %}

{% step %}
### Join!

Simply select the server you've added and press Join Server
{% endstep %}
{% endstepper %}
{% endtab %}

{% tab title="Bedrock" %}
{% stepper %}
{% step %}
### Open the servers tab

* Enter the Servers tab
{% endstep %}

{% step %}
### Select Add Server

Type Soltide for the server name.

Type Play.Soltide.net for the IP

Leave the port as 19132
{% endstep %}

{% step %}
### Join!

Simply press save to server list and join!
{% endstep %}
{% endstepper %}
{% endtab %}
{% endtabs %}
