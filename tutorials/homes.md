# Homes

Homes are an easy way to save a location and teleport back to it at a later date!

{% stepper %}
{% step %}
### How to set a home

{% hint style="warning" %}
Homes cannot be set in the spawn world.
{% endhint %}

To set a home simply type:

`/sethome <home name>`

The home name must be unique, and should describe the purpose of the place, i.e. "Base".
{% endstep %}

{% step %}
### How to teleport to a home

{% hint style="info" %}
There is a 7-second delay for teleporting to homes when not within spawn.
{% endhint %}

Simply type:

`/home <home name>`

If you do not remember the names of your homes, type:

`/homes`

to list them.
{% endstep %}

{% step %}
### How to delete a home

Simply type:

`/delhome <home name>`
{% endstep %}
{% endstepper %}
