# Bug Abuse

## What is bug abuse?

Bug abuse is utilizing any unintended feature for personal gain, or to cause damage to the server. This can come in a variety of different factors, and it is up the the determination of staff to see what counts as bug abuse.

## Economy Abuse

Utilizing unintended features of the economy is considered bug abuse. An item must be transformed in a way before being sold. If an item can be crafted into something more valuable than the value of the items then it is unintentional. Even IF an item went through a transformation, if it's sell value is too high it maybe result it in being disabled or reduced in an attempt to curb inflation.&#x20;

## Abusing for gain

The utilization of any bug to kill another player, increase your balance, or spawn in items is disallowed and will result in a ban. Not sure if something is a bug? Ask yourself whether the implementation of what you're using seems intentional. If you're still not sure, ask a staff member.

## Abusing for fun

Utilizing bugs, even if not for personal gain, is still disallowed. These types of bugs include things like escaping the map or sending broken messages. These bugs, while they may appear minor, can cause a lot of damage and thus are not allowed to be used.

## Abusing to destroy

The utilization of a bug with intent to damage the server (Such as via crashing) is disallowed and will result in a ban and various other punishments. Do not attempt to cause damage to the server or any of Soltide's infrastructure.

## Reporting bugs

The only way to avoid a punishment for using a bug is to properly report it. To report a bug, create a ticket in the discord and detail the what caused it and what's the result. If possible, also provide screen recordings and screenshots. Please note that you many ONLY use the bug to detail it, you may NOT cause harm to any other players, you MUST disclose any items you've gained from it, and you MUST not disclose or involve other players with the bug. For example, if you found a way to escape the map, explaining how to escape the map or teleporting players outside of the map WILL result in a ban.&#x20;
